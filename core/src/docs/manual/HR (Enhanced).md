## HR (Enhanced)

The `<hr>` tag now comes in three styles.

- `---` : <hr>
- `===` : <hr class="thick">
- `***` : <hr class="bold">

The supplied stylesheets have the following settings:

~~~
hr.bold { 
  border-width: 3px;
}

hr.thick { 
  border-width: 2px;
}
~~~
