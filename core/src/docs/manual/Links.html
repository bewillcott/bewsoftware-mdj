Links and Auto Links

What's New

  • Link "[id]s" are now case-sensitive. Having them as case-insensitive just left it open to all sorts of problems with no real benefit.
  • You can now add a new option: "!" - to set the link target to "_blank". This will open the link's location page in a new tab, and should make it the current window. This leaves your original page still open where you left it, making it easy to go back and forth between them, or just getting back to where you left off.
